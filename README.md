# ch2k Shop Management System

A comprehensive management system built with Next.js for Jewkkeryt Shop to streamline inventory, sales, and customer management operations.

## Features

- **Inventory Management**

  - Track jewelry items and stock levels
  - Manage product categories and specifications
  - Set pricing and discounts
  - Generate inventory reports

- **Sales Management**

  - Process sales transactions
  - Generate invoices and receipts
  - Track daily/monthly sales
  - Manage customer orders

- **Customer Management**
  - Store customer information
  - Track purchase history
  - Manage loyalty programs
  - Handle customer feedback

### Prerequisites

- Node.js 18.0 or later
- npm or yarn package manager
